🎯 What: Set a global request body size limit of 16MB in the Flask application configuration.
⚠️ Risk: Without this limit, the application is vulnerable to Denial of Service (DoS) attacks. An attacker could send arbitrarily large request payloads, exhausting memory and server resources.
🛡️ Solution: Configured `app.config['MAX_CONTENT_LENGTH'] = 16 * 1024 * 1024` immediately after Flask app initialization to enforce the limit. Also added a `test_security.py` unit test to ensure requests exceeding the limit correctly return a 413 HTTP status code.
